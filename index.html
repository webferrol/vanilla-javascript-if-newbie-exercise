Declaraciones condicionales

A menudo, cuando escribe código, desea realizar diferentes acciones para tomar diferentes decisiones.

Puede usar declaraciones condicionales en su código para hacer esto. A continuación puede ver un condicional en JavaScript.

if (condición) { // <- true

} else { // <- false

}
En JavaScript tenemos las siguientes declaraciones condicionales:
  • Utilice if para especificar un bloque de código para que sea ejecutado siempre y cuando si la condición es verdadera (true)
  • Use else para especificar un bloque de código que se ejecutará, si la misma condición es falsa
  • Use else if para especificar una nueva condición para probar, si la primera condición es falsa
  • Use switch (interruptor) para especificar muchos bloques alternativos de código para ejecutar
